# 크루셔틀 관리자 웹 페이지

버스 셔틀 운행을 관리하는 웹 기반 관리자 페이지입니다.

## 🚀 접속 방법

관리자 페이지: `/admin/`

## 🛠 기술 스택

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla)
- Supabase (데이터베이스)

## 📱 기능

- 실시간 운행 현황 모니터링
- 기사별 운행 통계
- 소속별 관리
- 엑셀 보고서 다운로드
- 인원보고 관리

## 🔧 설정

Supabase 연결 정보는 `admin/config.js` 파일에서 확인할 수 있습니다.

## 📄 라이선스

이 프로젝트는 내부 사용을 위한 것입니다.
